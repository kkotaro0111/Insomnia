# Insomnia
お前が寝るから俺ら眠れないじゃないか！！

You sleep, we can't sleep...

# What is this?
Minecraft用データパックです。
マルチサーバーで、複数人がプレイしているときに、全員寝ないと夜が明けないのはちょっと大変だよねーということで、一人でも寝たら朝になればよくね？ってなって結果作ったものです。
つまり「誰か一人でもベッドで寝たら、朝になるよ」と言うものです。

This is a databack for Minecraft.
The night will break when go to bed even one person.
